# Capturas de Pantalla

Esta carpeta contiene las capturas de pantalla de la aplicación.

## Estructura recomendada

- `login.png` - Página de inicio de sesión
- `products.png` - Página de gestión de productos
- `report.png` - Pagina de ventas y Reporte de ventas

## Cómo agregar capturas

1. Toma capturas de pantalla de la aplicación en funcionamiento
2. Guarda las imágenes en esta carpeta con los nombres sugeridos
3. Las imágenes se referencian automáticamente en el README principal
